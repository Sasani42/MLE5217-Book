If you are new to python or new to programming, you should start at the begining, and go through in sequence. This gives you an introduction to programming in python and then gives you instructions on downloading and installing all the requirements to get you going in python. 

Installation and setting up is fairly straight forward but should you run into any issues you could either contact me at `sjaya@nus.edu.sg` or look up online tutorials on how to overcome your specific issues. The good thing with Python is that because it is widely used, there's plenty of resources to help you out.

Once you run the installation and print your first "Hello world" check, you are ready to follow the course.
The lessons are embedded as videos, accompanying text details and followed by self-check questions. 

:::{tip}
As you follow the video, open your notebook and follow along. Pause the video and try the exercises yourself. Python is a skill you can only master when you do it yourself!
:::

Use the sidebar to navigate. A good place to start is **Week 0 → Getting things ready**.
