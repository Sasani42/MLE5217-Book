# MLE5217 — Foundations of Machine Learning for Materials Science

Test test test!

This is the course book for MLE5217 for the academic year 2026/27 Semester I. As you progress through the course new content will be added each week. 

The main machine learning topics that we will be looking at are:
* Introduction to ML - Python & Linear Algebra
* Statistics
* Linear Regression & Polynomial Regression
* Data Cleaning
* Clustering
* Featurization
* Bias Vs Variance & Cross Validation
* Hyper-parameter Tuning
* Data Ethics & Algirithmic Bias
* Ridge, Lasso & ElasticNet Regression
* Gaussian Process Regression
* Support Vector Machines
* Decision Trees & Random Forests
* ML in Materials Science & Materials Databases
* Introduction to Neural Networks 


Starting from week 2 - Each week has three pages:

- a **“What to expect”** preview to read *before* the lecture, and
- the **lecture** itself — readable notes with code you can run and links to any additional materials
- a quick **self-check quiz** to test your understanding of the topic.

:::{tip}
You don't need to install anything to read this book. Where you see a grey code
cell, that is real Python from the lecture — and (once live execution is switched
on) you'll be able to run and edit it right here in your browser.
:::

Use the sidebar to navigate. A good place to start is **Week 0 → Getting things ready**.
